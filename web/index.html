使用帮助

首次使用步骤

  1. 使用用户名和密码登录系统
  2. 获取 Telegram API ID 和 API Hash
  3. 在配置区域填写 API 信息和手机号
  4. 输入收到的验证码完成登录
  5. 上传文件：拖放文件或点击"选择文件"按钮
  6. 下载文件：在文件列表中找到文件，点击下载按钮

文件列表功能

文件列表页面可以查看和管理所有已上传的文件，包括：

  • 按文件类型筛选（全部、图片、视频、音频、文档）
  • 搜索文件名称
  • 按不同条件排序（日期、名称、大小）
  • 下载、分享和删除文件
  • 批量操作多个文件

常见问题

如何处理大文件上传？

工具会自动分割大文件，你可以在上传设置中增加并行线程数提高速度

文件保存在哪里？

文件会保存在你的 Telegram 账号的"已保存消息"中，只有你能看到

上传失败怎么办？

检查网络连接，确认 Telegram 账号正常，然后重试上传
